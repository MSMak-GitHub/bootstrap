group: utilities

**Не забудьте включить текст для программ чтения с экрана**, как мы это сделали с `aria-label`.

{{< example >}}
<button type="button" class="close" aria-label="Закрыть">
  <span aria-hidden="true">&times;</span>
</button>
{{< /example >}}
